Project 2:Investigate no show medical appointments in Brazil¶

Introduction¶

In this project we investigate a dataset of 100K medical appointments in Brazil. The dataset contains various characteristics for each patient and whether or not he showed up to the appointment. Exploring the data we aim to answer the following questions: 1) Which is the show up and no-show rate, for the given dataset? 2) Does waiting time and SMS receipt affect show or no show to an appointment? 3) Are there any specific attributes,like for example age and gender, influence no shows?

def get_date(datetime_object):
    '''This function returns the date part of a datetime'''
    return datetime_object.date()

## Calculate Lead Time From the scheduled appointment till appointment date/ timedelta64[ns] format
no_show['Lead_days'] = (
    no_show.Appointment_day.apply(get_date)
    - no_show.Scheduled_day.apply(get_date)
    )
# Transform Lead_days as integer
#no_show['Lead_days']=no_show['Lead_days']/timedelta (days=1)
no_show.Lead_days = (

3.Conclusion¶

Examining the medical appointments dataset we tried to understand the severity of the problem and if there are specific attributes or factors that may affect the show or no- show to the appointment. We used descriptive statistics, aiming to summarize results for this sample.One limitation we faced is that the data are referred to a 5 month period, creating difficulty to draw clear statements. Although the dataset contains only 110k appointments, it seems that age,lead time and SMS receipt are factors that influence a show up. For example, younger ages tend to miss appointments in a greater rate than older ages.Small lead times tend to be more efficient.Last but not least, when patients receive SMS seems to remember the appointment, so the no show rate is lower, even if the lead time is high.
